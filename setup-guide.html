📋 システム概要

このシステムは、備前丸停泊内覧会のために開発された、QRコードベースの参加者管理システムです。完全無料で使用でき、300〜800名規模のイベントに対応しています。

🗂️ ファイル構成

  • index.html - エントリーフォーム（参加者用）
  • ticket.html - デジタルチケット表示
  • scan.html - スキャン管理画面（係員用）
  • dashboard.html - 管理ダッシュボード（管理者用）
  • README.md - 詳細ドキュメント
  • setup-guide.html - このファイル

🚀 セットアップ手順

1 ファイルを解凍

bizen-maru-system.zip を解凍して、すべてのHTMLファイルを確認してください。

2 ホスティングサービスを選択

以下のいずれかの無料ホスティングサービスを使用してください：

サービス URL 難易度
Netlify（推奨） netlify.com ★☆☆
Vercel vercel.com ★☆☆
GitHub Pages pages.github.com ★★☆
3 ファイルをアップロード

選択したホスティングサービスに、すべてのHTMLファイルをアップロードします。

💡 Netlifyの場合
  1. Netlifyにログイン
  2. 「Sites」→「Add new site」→「Deploy manually」
  3. 解凍したフォルダをドラッグ＆ドロップ
  4. 自動的にURLが発行されます（例: https://bizen-maru-xxx.netlify.app）
4 エントリー用QRコードを生成

発行されたURLに /index.html を追加して、QRコードを生成します。

  1. QRコード生成サイトにアクセス
  2. URL欄に https://あなたのURL/index.html を入力
  3. QRコードをダウンロード
  4. A4サイズで印刷して会場に設置
5 係員用端末の準備

スキャン用の端末（スマホまたはタブレット）2台を準備します。

  • ブラウザで https://あなたのURL/scan.html を開く
  • カメラアクセスを許可
  • ホーム画面に追加（ブックマーク）して、素早くアクセスできるようにする
6 管理者用ダッシュボードの設定

管理者用PCまたはタブレットで、ダッシュボードを開きます。

  • ブラウザで https://あなたのURL/dashboard.html を開く
  • リアルタイムで統計が自動更新されます（10秒ごと）
  • 必要に応じてCSVエクスポート機能を使用

📅 当日の運用フロー

参加者側の流れ
  1. 会場のQRコードをスマホでスキャン
  2. エントリーフォームに必要事項を入力
  3. 送信すると、無料乗船チケット（QRコード）が表示される
  4. スクリーンショットを保存（推奨）
  5. 乗船時に係員にQRコードを提示
  6. 下船時に再度QRコードを提示
係員側の流れ
  1. scan.htmlを開いてスタンバイ
  2. 乗船時：「乗船モード」でQRコードをスキャン
  3. 画面に「乗船を記録しました」と表示されることを確認
  4. 下船時：「下船モード」に切り替えてスキャン
  5. 滞在時間が自動計算されて表示される
管理者側の作業
  1. dashboard.htmlで全体の状況を監視
  2. 船内人数をリアルタイムで確認（定員管理）
  3. 必要に応じて係員に指示
  4. イベント終了後、CSVエクスポートでデータを保存

🔧 トラブルシューティング

Q: カメラが起動しない

A: ブラウザのカメラアクセス許可を確認してください。設定→プライバシー→カメラで、ブラウザにカメラの使用を許可してください。それでも解決しない場合は、手動入力機能をご利用ください。

Q: QRコードが読み取れない

A: 明るい場所で、QRコードを画面中央に大きく表示してください。それでも読み取れない場合は、画面下部の「手動入力」欄にチケットIDを入力してください。

Q: オフライン時の対応

A: オフライン表示が出た場合でも、手動入力で記録できます。データは端末内に一時保存され、オンライン復帰後も保持されます。

Q: データのバックアップ

A: イベント中に定期的に、dashboard.htmlから「CSVエクスポート」ボタンをクリックして、データをダウンロードしてください。イベント終了後も必ず実行してください。

⚠️ 重要な注意事項

データの保存場所

現在の実装では、データは各端末のブラウザに保存されます。複数端末間でのリアルタイム同期は行われませんので、以下の点にご注意ください：

  • スキャン用端末2台は、それぞれ独立してデータを記録します
  • ダッシュボードは、開いている端末内のデータのみを表示します
  • 各端末から定期的にCSVエクスポートすることを推奨します
個人情報の取り扱い

参加者の個人情報が含まれますので、以下の点にご注意ください：

  • イベント終了後、データを安全に保管してください
  • 不要になったデータは適切に削除してください
  • エクスポートしたCSVファイルの取り扱いに注意してください
