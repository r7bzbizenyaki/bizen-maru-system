# 備前丸 内覧会管理システム

備前丸停泊内覧会向けのQRコードベース予約・乗船管理システムです。完全無料で使用できます。

## 📋 システム概要

### 機能
- ✅ エントリーフォーム（参加者情報入力）
- ✅ デジタルチケット生成（QRコード付き）
- ✅ 乗船・下船時のスキャン機能
- ✅ 参加者データベース管理
- ✅ リアルタイム統計ダッシュボード
- ✅ CSVエクスポート機能
- ✅ オフライン対応

### 対応規模
- 想定参加者数: 300〜800名
- 同時スキャン端末: 2台
- 当日受付対応

## 🚀 使用方法

### 1. エントリーフォーム（index.html）
**対象**: 参加者
**用途**: 会場でQRコードをスキャンして表示

参加者は以下の情報を入力します：
- お名前（必須）
- 郵便番号（必須）
- 住所（必須）
- 電話番号（必須）
- メールアドレス（必須）
- 年代（必須）
- 性別（必須）

送信後、自動的に無料乗船チケットが表示されます。

### 2. デジタルチケット（ticket.html）
**対象**: 参加者
**用途**: 乗船時に係員に提示

- 固有のチケットIDとQRコードが表示されます
- スクリーンショット保存推奨
- 乗船時・下船時に係員がスキャンします

### 3. スキャン管理画面（scan.html）
**対象**: 係員（2名・2台の端末）
**用途**: 乗船・下船の記録

**機能**:
- 📷 カメラでQRコードをスキャン
- 🚢 乗船モード / 👋 下船モード 切替
- 🔢 手動入力（オフライン時）
- 📊 リアルタイム統計表示
  - 乗船者数
  - 下船者数
  - 船内人数
  - 平均滞在時間

**操作手順**:
1. ブラウザでscan.htmlを開く
2. カメラアクセスを許可
3. モード（乗船/下船）を選択
4. 参加者のQRコードをスキャン
5. 画面に結果が表示される

### 4. 管理ダッシュボード（dashboard.html）
**対象**: 管理者
**用途**: データ分析・エクスポート

**機能**:
- 📊 総合統計（エントリー数、乗船数、下船数、平均滞在時間）
- 📈 年代別・性別分析（グラフ表示）
- 📋 参加者リスト（詳細一覧）
- 📥 CSVエクスポート
- 🔄 リアルタイム自動更新（10秒ごと）

## 📱 QRコード生成方法

### エントリーフォーム用QRコード

1. **オンラインQRコード生成サイトを使用**
   - https://www.qrcode-monkey.com/
   - https://www.the-qrcode-generator.com/

2. **URLを入力**
   ```
   https://あなたのドメイン.com/index.html
   ```

3. **QRコードを生成・ダウンロード**

4. **会場に設置**
   - A4サイズで印刷
   - 「備前丸内覧会エントリーはこちら」と記載
   - 受付エリアに配置

## 💾 データ保存について

### 現在の実装（ローカルストレージ）
- データは各端末のブラウザに保存されます
- 同じ端末・同じブラウザでのみデータを共有
- ページを閉じてもデータは保持されます

### 制限事項
- 複数端末間でのリアルタイム同期は未対応
- バックアップは手動（CSVエクスポート）

### 今後の拡張案（Google Sheets連携）
より本格的な運用をご希望の場合、Google Sheets APIを使用して複数端末でのリアルタイム同期が可能です。

**必要な作業**:
1. Google Cloud Platformでプロジェクト作成
2. Google Sheets API有効化
3. サービスアカウント作成
4. JavaScriptコードにAPI連携処理を追加

## 🔌 オフライン対応

### オフライン時の動作
- ✅ エントリーフォーム: データを一時保存
- ✅ スキャン管理: 手動入力で記録可能
- ✅ ローカルストレージに保存されます

### オンライン復帰時
- 自動的にオフライン表示が消えます
- 保存されたデータは端末内に保持されます
