# 備前丸内覧会システム - 簡易インストールガイド

## 🎯 最も簡単な使用方法（インターネット不要）

### 方法1: ローカルで直接使用（推奨）

**セットアップ:**
1. ZIPファイルを解凍
2. 完了！すぐに使えます

**使用方法:**

#### 📱 参加者用エントリーフォーム
- **ファイル:** `index.html`
- **使い方:** ダブルクリックで開く
- **設置:** 受付用PC/タブレットで表示

#### 🚢 係員用スキャン画面
- **ファイル:** `scan.html`
- **使い方:** スマホ/タブレットのブラウザで開く
- **人数:** 2台準備

#### 📊 管理者用ダッシュボード
- **ファイル:** `dashboard.html`
- **使い方:** PCのブラウザで開く
- **機能:** リアルタイム統計・CSVエクスポート

## 🌐 方法2: LAN内で共有（会場内ネットワーク）

会場内のWi-Fi経由で、複数端末から同じURLでアクセスしたい場合：

### Windowsの場合

1. **ZIPファイルを解凍**
2. **start-server.bat をダブルクリック**
3. **表示されたIPアドレスをメモ**（例: 192.168.1.100）
4. **他の端末から以下でアクセス:**
   - エントリー: `http://192.168.1.100:8080/index.html`
   - スキャン: `http://192.168.1.100:8080/scan.html`
   - ダッシュボード: `http://192.168.1.100:8080/dashboard.html`

### Mac / Linuxの場合

1. **ZIPファイルを解凍**
2. **ターミナルを開く**
3. **以下を実行:**
   ```bash
   cd 解凍したフォルダのパス
   chmod +x start-server.sh
   ./start-server.sh
   ```
4. **表示されたIPアドレスをメモ**
5. **他の端末から上記URLでアクセス**

### IPアドレスの確認方法

**Windows:**
1. コマンドプロンプトを開く
2. `ipconfig` を実行
3. 「IPv4アドレス」をメモ（例: 192.168.1.100）

**Mac:**
1. システム環境設定 → ネットワーク
2. 接続中のWi-Fiを選択
3. IPアドレスをメモ

1. https://www.qrcode-monkey.com/ にアクセス
2. 以下のURLを入力:

3. QRコードを生成・ダウンロード

3. 送信後、チケット画面をスクリーンショット

## ❓ よくある質問

### Q: インターネット接続は必要ですか？
**A:** 不要です。完全オフラインで動作します。

### Q: スマホで開くにはどうすればいいですか？
**A:** 
- ZIPを解凍したフォルダをスマホに転送
- ファイルマネージャーで `scan.html` を開く
- または、LAN共有でURLからアクセス

### Q: データはどこに保存されますか？
**A:** 各端末のブラウザ内に保存されます（ローカルストレージ）。ページを閉じても消えません。

### Q: 複数端末でデータを共有できますか？
**A:** 現在のバージョンではリアルタイム同期はありません。各端末から個別にCSVエクスポートして、後でExcelで統合してください。

### Q: カメラが起動しない
**A:** ブラウザのカメラアクセス許可を確認してください。または、手動入力機能を使用してください。

## 📞 トラブル時の対応

問題が発生した場合は、以下を確認してください：

1. ✅ すべてのファイルが同じフォルダにあるか
2. ✅ ブラウザは最新版か（Chrome, Edge, Safari推奨）
3. ✅ JavaScriptが有効になっているか
4. ✅ カメラアクセス許可が与えられているか

それでも解決しない場合は、setup-guide.html を開いて詳細なトラブルシューティングをご確認ください。
